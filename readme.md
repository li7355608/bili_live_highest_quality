# 哔哩哔哩直播 - 自动切换最高画质

一个用于自动切换哔哩哔哩直播间画质为最高画质的用户脚本，直播必须要看最高画质狠狠滴薅叔叔的服务器带宽。

## 功能特性

- ✅ 自动检测并切换到最高画质
- ✅ 智能等待播放器加载完成
- ✅ 切换完成后自动卸载，不常驻内存
- ✅ 支持嵌套页面自动跳转

## 安装方法

1. 安装用户脚本管理器：

2. 安装脚本：[一键安装](https://greasyfork.org/zh-CN/scripts/563041-哔哩哔哩直播-自动切换最高画质)
   

## 使用方法

1. 打开任意哔哩哔哩直播间页面：`https://live.bilibili.com/*`
2. 脚本会自动运行，等待播放器加载完成后自动切换到最高画质
3. 切换完成后脚本会自动卸载，不会影响后续的手动切换画质

## 技术说明

- **执行间隔**：脚本启动后每秒检测一次画质状态，自动切换为当前直播间最高清晰度
- **稳定检测**：连续5次检测到最高画质后认为切换成功，切换完成后自动清理
- **超时机制**：播放器加载最多等待30秒，加载失败后不在尝试加载
- **嵌套页面处理**：脚本会自动处理嵌套页面，如：`https://live.bilibili.com/live/?room_id=*` 需手动开启此功能
